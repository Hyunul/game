# Episode 2 「궤짝 속 여름」 — 추리 방탈출 디자인

날짜: 2026-07-02
상태: 사용자 승인 대기
전편: 「기억의 상자」 (docs/superpowers/specs/2026-06-11-memory-box-escape-design.md)

## 1. 개요

다락방 허브의 **궤짝**에서 시작하는 두 번째 에피소드. 장르는 **추리 방탈출**(살인 의혹 미스터리, 비노골 — 시신·폭력 묘사 없음, 사건은 기록·회상·대사로만 재구성). 예상 플레이타임 **약 60분**, 난이도는 본편보다 반 단계 위, 퍼즐별 2단계 힌트 유지.

## 2. 스토리

궤짝의 유품 세 가지 — 빛바랜 흑백 가족사진, **1978년 8월 16일자 신문 스크랩**("15일 밤 저수지에서 청년 익사… 타살 의혹"), **11시 40분에 멈춘 회중시계**. 신문 속 청년은 주인공의 **큰아버지**. 마을에는 "동생(주인공의 아버지)과 크게 다툰 다음 날 죽었다"는 소문이 평생 따라다녔고, 아버지는 그 일을 입에 올린 적이 없다. 회중시계의 태엽을 감는 순간, 주인공은 1978년 여름의 큰집으로 들어간다.

**진실(엔딩 반전)**: 그날 밤 저수지에 빠진 것은 **동생**이었다. 형은 맨몸으로 뒤따라가 동생을 물 밖으로 밀어 올리고 자신은 나오지 못했다. 전날의 "다툼"도 동생의 보름밤 낚시를 막으려던 것. 아버지는 "내가 형을 죽게 했다"는 죄책감을 소문 뒤에 숨기고 살아왔다. 주인공이 재구성한 진실이 아버지에게 닿으며, 살인 의혹이 **희생과 형제애**의 이야기로 뒤집힌다.

## 3. 핵심 메커니즘 — 시점 전환

- 인벤토리의 **회중시계**를 선택해 사용(아무 장면에서나)하면 **현재(먼지 쌓인 폐가) ↔ 과거(1978년 8월 15일 낮~저녁)** 전환. 전환 시 물결 연출 + 색조 변화(현재: 차갑고 바랜 톤 / 과거: 따뜻한 여름 톤)
- 인벤토리는 시점 간 공유(회중시계의 힘이라는 설정, 프롤로그에서 내레이션 한 줄로 처리)
- 같은 공간의 과거/현재 **차이를 관찰**하는 것이 추리의 핵심 (기울어진 액자, 페인트로 덮인 눈금, 사라진 도구)
- 최종 단계에서만 과거가 **밤**으로 바뀐다 (조건 충족 시 이벤트)

## 4. 페어플레이 원칙

1. 모든 답은 장면 안에서 관찰 가능한 **단서 2개 이상이 수렴**해서 도출된다.
2. 자물쇠·비밀번호에는 **그 인물이 실제로 골랐을 법한 서사적 이유**가 있다 (기일, 키, 보관 습관).
3. 단서는 증거를 겸한다 — 퍼즐 하나를 풀면 사건 정황도 하나 밝혀진다.
4. 힌트 2단계로 어떤 플레이어도 막히지 않게 한다 (1단계 방향 / 2단계 거의 정답).

## 5. 공간과 퍼즐 — 단서 → 추론 → 답

공간: **사랑방(서재) / 안방과 마루 / 헛간과 마당** × 2시점, 최종 **저수지 가는 길**(과거의 밤).

### 프롤로그 (다락방)
궤짝 클릭 → 유품 3종 획득(가족사진·신문 스크랩·회중시계) + 짧은 내레이션. 회중시계 사용 → 큰집(현재)으로 진입.

### 진실 조각 (5개 — 최종 퍼즐의 재료)

| 조각 | 증거 | 밝혀지는 정황 |
|---|---|---|
| ① 형의 부치지 못한 편지 | "동생을 탓하지 마세요. 그 아이 잘못이 아닙니다" | 전날의 '다툼'은 동생의 밤낚시를 막으려던 것 |
| ② 동생의 쪽지 | "8월 15일 보름, 밤낚시. 형한테는 비밀" | 그날 밤 저수지에 간 사람은 동생 |
| ③ 장비의 정황 | 도구 걸이 윤곽선 대조 — 낚싯대 1개·양동이만 없음 | 떠난 사람은 한 명뿐, 형은 맨몸으로 뒤따랐다 |
| ④ 멈춘 회중시계 | 물때 + 11:40 + 안쪽 새김 "아우와 함께 — 아버지가" | 형이 물에 들어간 시각, 시계는 형의 유품 |
| ⑤ 어머니의 일기 | "16일 새벽. 막내가 젖어 돌아왔다. 옷을 빨아 널었다. 아무에게도 말하지 않았다." | 동생은 살아 돌아왔다 — 물에 빠졌던 건 동생 |

### 최종 — 저수지 가는 길 (과거의 밤)

- **진입 조건**: 진실 조각 5개 + 랜턴 점등 → 과거가 밤으로 바뀌는 이벤트: 마당에서 담을 넘는 동생의 뒷모습 목격 → 따라간다
- **타임라인 재구성 퍼즐**: 저수지 앞에서 조각 5개가 사건 카드 5장으로 정리됨. **시간 순서대로 배열**:
  ① 전날 — 형제의 다툼(편지) → ② 15일 밤 — 동생 몰래 출발(쪽지) → ③ 형이 알아채고 맨몸으로 뒤따름(장비) → ④ 밤 11:40 — 형이 동생을 밀어 올림(시계) → ⑤ 16일 새벽 — 동생 홀로 귀가(일기)
- 오답 배열은 흔들림 + "무언가 앞뒤가 맞지 않는다" / 정답 시 **실루엣 회상 연출**: 보름달 저수지, 물결, 두 실루엣 — 한 손이 한 손을 밀어 올린다 (비노골, 음악과 텍스트 중심)
- **에필로그**: 현재의 다락방. 주인공이 아버지에게 전화를 건다. "아버지… 큰아버지 이야기, 이제 알 것 같아요." 짧은 답신 텍스트와 여운. 궤짝 명패가 「궤짝 속 여름 — 완료」로 변경

## 6. 기술 구조 (본편 무변경 원칙)

- **에피소드 일반화(소폭 리팩터링)**: `lib/episode.ts`에 EpisodeConfig(퍼즐 배열·아이템·저장 키·최종 퍼즐 id) 정의. 리듀서를 config 주입형으로 일반화하되 **본편 동작·테스트는 그대로 통과**해야 한다
- **시점**: GameState에 `era: 'past' | 'present'` 필드(ep1은 항상 'present'), `TOGGLE_ERA` 액션. 퍼즐 데이터에 선택적 `era` 조건
- **저장 분리**: ep2는 `memory-box-save-ep2` 키. 다락방 궤짝 카드에 "시작하기 / 이어하기" 표시
- **장면**: `components/scenes/ep2/` — Sarangbang.tsx, Anbang.tsx, Heotgan.tsx, Reservoir.tsx (과거/현재는 같은 컴포넌트 안에서 era 분기 렌더링). 본편 컨벤션 재사용(hotspot, Keypad, Narration, 회상 연출, 두 번 탭)
- **아트**: 본편과 같은 플랫 SVG. 현재 = 차가운 회갈색 톤, 과거 = 따뜻한 여름 톤, 밤 = 남색+보름달
- **음악**: 과거/현재 별도 BGM 2곡 + 밤 테마 1곡 (Web Audio 합성, audio.ts 확장)
- **진입/허브**: 궤짝 소개 카드의 Coming Soon → "시작하기" 버튼. ep2 플레이 중 상단바 ⚙️는 "허브로 나가기 / 처음부터" 선택지
- **테스트**: ep2 전체 플레이스루 리듀서 테스트, era 전환·조건 게이트 테스트, 본편 회귀 테스트 유지
