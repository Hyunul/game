# 「기억의 상자」 — 감동·추억 반응형 웹 방탈출 게임 디자인

날짜: 2026-06-11
상태: 사용자 승인 대기

## 1. 개요

"감동"과 "추억"을 키워드로 한 반응형 웹 방탈출 게임. 플레이어 보편의 향수(90~2000년대 한국의 어린 시절)를 자극하는 세 공간을 순회하며 퍼즐을 푼다. Next.js로 제작하여 Vercel에 배포한다.

- 플레이 분량: 20~30분 (미디엄 난이도, 방마다 퍼즐 4~5개, 2단계 힌트 제공)
- 아트 스타일: 따뜻한 플랫 일러스트 (SVG 벡터, 코드로 직접 작화)
- 사운드: Web Audio API 합성 BGM(방마다 상이) + 효과음, 음소거 버튼 제공
- 반응형: 데스크톱 마우스 + 모바일 터치 모두 지원

## 2. 스토리

어른이 된 주인공이 이삿짐을 정리하다 다락에서 낡은 종이 상자를 발견한다. 상자 안 물건을 집어들 때마다 그 시절의 공간으로 빨려 들어간다.

- **프롤로그**: 다락방. 상자를 열고 물건을 순서대로 선택 (사진 → 필통 → 뽑기 캡슐, 순서 고정)
- **방 1 — 옛날 우리 집 안방** (가족사진)
- **방 2 — 초등학교 교실** (필통)
- **방 3 — 학교 앞 문방구** (뽑기 캡슐)
- **에필로그**: 기억 조각 3개가 상자 안에서 합쳐지고 잔잔한 메시지와 함께 엔딩. "탈출"의 의미가 사실은 "추억을 되찾는 것"이었다는 여운.

각 방의 탈출 조건: 퍼즐을 풀어 최종 잠금장치를 열면 '기억 조각'을 얻고 다락방으로 귀환.

## 3. 방별 퍼즐 설계

포인트 앤 클릭 방식. 사물을 클릭(터치)해 조사하고, 아이템을 인벤토리에 넣고, 조합·사용해 퍼즐을 푼다.

### 방 1 — 옛날 우리 집 안방 (5개)
1. **벽걸이 달력**: 동그라미 쳐진 날짜 발견 → "우리 가족 기념일" 메모
2. **다이얼 전화기**: 달력의 날짜를 다이얼로 돌리면 수화기에서 단서 ("TV를 켜보렴")
3. **브라운관 TV**: 가족사진 뒷면에 적힌 채널로 맞추면 화면에 숫자 힌트
4. **반짇고리**: 장롱 위에 있어 닿지 않음 → **효자손** 아이템으로 끌어내림 (아이템 사용)
5. **자개장 서랍 (최종)**: TV 숫자 자물쇠 → '기억 조각 1: 엄마의 자장가'

### 방 2 — 초등학교 교실 (5개)
1. **시간표**: 색칠된 과목 칸 → 요일+교시 좌표
2. **사물함**: 좌표 번호의 사물함에서 **풍금 악보** 획득
3. **풍금**: 악보대로 건반 연주(실제 소리 발생) → **분필** 획득
4. **칠판**: 분필로 문지르면 지워진 글씨 → "교환일기는 화분 아래에"
5. **교탁 서랍 (최종)**: 교환일기의 비밀번호 → '기억 조각 2: 친구의 약속'

### 방 3 — 학교 앞 문방구 (5개)
1. **불량식품 진열대**: "300원어치 골라줘" 쪽지에 맞는 과자 조합 클릭
2. **돼지저금통**: 과자 퍼즐 보상으로 **100원 동전** 획득
3. **오락기**: 동전 투입 → 두더지잡기 미니게임 (5마리) → 비밀 코드
4. **종이인형 책**: 코드 페이지에서 **뽑기용 동전** 발견
5. **뽑기 기계 (최종)**: 동전을 넣고 돌리면 황금 캡슐 → '기억 조각 3: 백 원의 행복'

### 공통 시스템
- **인벤토리**: 하단 바 최대 6칸. 아이템 탭 → 사물 탭으로 사용
- **힌트**: 퍼즐별 2단계 (1단계 방향 제시 / 2단계 거의 정답)
- **저장**: localStorage 자동 저장, 첫 진입 시 "이어하기" 제공
- **감동 연출**: 기억 조각 획득 시 회상 일러스트 + 내레이션 텍스트 + 음악

## 4. UI / 반응형

- **장면 = SVG 벡터** (viewBox 비율 유지, 모든 해상도에서 선명)
- **상단 바**: 방 이름 / 힌트 / 음소거 / 설정(처음부터)
- **하단 인벤토리** 고정
- **내레이션**: 하단 반투명 텍스트 박스, 탭하면 진행
- **모바일**: 좁은 화면에서 장면 좌우 스와이프 패닝, 터치 영역 최소 44px
- **호버 대응 (모바일)**: 첫 탭 = 글로우 + 사물 이름, 두 번째 탭 = 조사

## 5. 마이크로 인터랙션

- **호버**: 황금빛 글로우 + scale 1.03 + 돋보기 커서 + 사물 이름 툴팁
- **클릭/탭**: 클릭 지점 파티클 리플 + squash & stretch 바운스 + 효과음
- **빈 영역 클릭**: 미세한 화면 흔들림 + 먼지 파티클
- **아이템 획득**: 포물선으로 인벤토리로 날아가 안착, 칸 반짝임
- **정답**: 따뜻한 빛 펄스 + 차임벨 / **오답**: 좌우 흔들림 + 낮은 소리
- **기억 조각 획득**: 빛 입자가 화면 가득 떠오르며 회상 장면 전환
- **방 전환**: 물결 일렁임 페이드
- 구현: CSS 트랜지션 + SVG 애니메이션 + 가벼운 파티클(DOM/Canvas), 외부 라이브러리 없음

## 6. 기술 아키텍처

**스택: Next.js 15 (App Router) + TypeScript + Vercel**. 게임은 전부 클라이언트 사이드 — 서버 API 없음.

```
app/
  layout.tsx          # 전역 레이아웃, 폰트, 메타데이터
  page.tsx            # 게임 진입점 ("use client")
components/
  GameShell.tsx       # 상단바 + 장면 + 인벤토리 + 내레이션 골격
  scenes/
    Attic.tsx         # 프롤로그/에필로그 다락방
    Room1Home.tsx     # 안방
    Room2Class.tsx    # 교실
    Room3Store.tsx    # 문방구
  Inventory.tsx
  HintModal.tsx
  MemoryScene.tsx     # 기억 조각 획득 연출
lib/
  gameState.ts        # useReducer 기반 게임 상태
  puzzles.ts          # 퍼즐 정의 데이터 (정답, 해금 조건, 힌트)
  audio.ts            # Web Audio API BGM 합성 + 효과음
  save.ts             # localStorage 저장/불러오기
```

**설계 결정:**
- **상태 관리**: 단일 상태 객체(현재 방, 풀린 퍼즐 집합, 인벤토리, 플래그)를 `useReducer` + Context로 관리. 외부 상태 라이브러리 없음.
- **퍼즐 = 데이터**: 퍼즐 로직은 `puzzles.ts`에 선언적으로 정의, 장면 컴포넌트는 표시·클릭만 담당.
- **사운드**: 외부 음원 파일 0개, 코드 합성. 첫 사용자 인터랙션 후 재생 시작(자동재생 정책 대응).
- **저장**: 상태 변경 시 localStorage 직렬화.

## 7. 테스트 / 검증

- **단위 테스트 (Vitest)**: 퍼즐 해금 로직·상태 전이 (예: "TV 코드 입력 → 자개장 해금"), 저장/복원 직렬화
- **수동 검증**: 빌드 후 전 구간 플레이스루 (데스크톱 + 모바일 뷰포트), 힌트 시스템, 이어하기
- **배포**: Vercel — GitHub 연동 또는 `vercel` CLI. 정적 콘텐츠로 무료 플랜 충분.

## 8. 범위 제외 (YAGNI)

- 다국어 지원 (한국어만)
- 서버 저장 / 계정 / 랭킹
- 모바일 앱 패키징
- 방 추가 확장 (단, 퍼즐 데이터 구조는 자연히 확장 가능)
